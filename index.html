BitcoinXT

Bitcoin XT is an implementation of a full node that embraces Bitcoin's original vision of simple, reliable, low-cost transactions for everyone in the world.

BitcoinXT is based upon the source code of Bitcoin Core, and is built by taking the latest stable Core release, applying a series of patches, and then doing deterministic builds so anyone can check the downloads correspond to the source code. BitcoinXT implements BIP 101, which introduces bigger blocks for the network.

Bitcoin Core

Bitcoin core is a standard node that helps relay transactions and store the blockchain around the world

Bitcoin nodes form a simple broadcast network to propagate transactions and blocks. The more nodes there are, the stronger the network becomes.
